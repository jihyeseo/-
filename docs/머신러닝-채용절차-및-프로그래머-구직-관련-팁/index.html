    머신러닝 채용절차 및 프로그래머 구직 관련 팁

    제가 경험한 데이터 사이언스/머신러닝 채용절차는요, 처음에 인사과 통화 1번, 기술면접 전화 1번, (핸드폰으로 오기도 하고 비디오 오디오 채팅하기도 하고요) 그리고 방문면접 1번 정도가 있고요. 그 단계단계 사이에 옵셔널하게 들어있는게, 1. 데이타 셋 던져주면서 이거로 특정 문제 풀어와, 하면서 1주일 준다거나. (머신러닝 챌린지라고 하더라고요) 보통 시간을 자기가 정할 수 있으니까, 편한 날짜 맞춰서 미리 시간 1주일 비워놓고 하면 되요. 2. 코딩챌린지, 파이썬 도커 등을 준비하시면 될 것 같고, 온라인 상에서 화면 쉐어 하면서 설명하면서 문제 푸시는 거요. 보통 1시간 이하. 3. 객관식 문제 머신러닝 상식 테스트 문제은행 풀기, 혹은 stepstone, xing, linkedin, monster, indeed, glassdoor, stackoverflow, moberries, talent.io, honeypot, angel.co, 등에 가서 잡 공고 보시면 될 것 같아요. 거기에 구직 프로필 꾸미시면 더 좋고요. 사진도 프로페셔널하게 하나 찍어 프사 올리시고요. 작고 영세한 회사라면, 이미 베를린에 와 있다는 사실을 장점으로 생각할 것 같기는 해요. (비용과 시간을 아끼기 위해) 혹시 나이가 아직 젊으셔서 워킹홀리데이 비자 등이 되어 취업비자가 금방 나오면 또 장점이겠네요. 근데 큰회사 (아마존, 딜리버리 히어로, 및 많이 들어보신 회사 이름 등등) 가시면 이미 베를린 와 있지 않더라도 이사비용까지 주면서 채용할 수도 있으니 걱정 안 할 것 같아요.
